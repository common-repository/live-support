Adds call button to your site. Your vistors can make web calls right away from your site.

== Description ==

This plugin helps you to embed the call button in your website without changing your theme files.

You need API key from <a href="https://calltosupport.com" target="_blank">https://calltosupport.com</a>. After you have kept the API key, your vistors can make web to web voice calls for free. 

Note : You need to login to operator panel of <a href="https://calltosupport.com" target="_blank">https://calltosupport.com</a> to receive the calls.

For more details visit <a href="https://calltosupport.com" target="_blank">https://calltosupport.com</a>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the setting page of the plugin from Settings > Call To Support and keep the API key you got from your calltosupport.com account dashboard.
